## Description

A pointing application with nestJs

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Running the app with docker
```bash
$ docker compose up

```
## To see the documetation for api

```bash
$ localhost:3000/api

```
